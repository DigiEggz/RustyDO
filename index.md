## Getting started

### Requirements

+ [Cargo](https://www.rust-lang.org/tools/install): A rust compiler.

### Quick start: Run the server

1. Clone the repository
```bash
git clone https://github.com/jidoc01/RustyDO
```

2. Run the server
```bash
cargo run
```

<small>Done! Now, RustyDO is ready.</small>
